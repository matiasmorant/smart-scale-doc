``converter.py``
================

These are some of the functions defined in the module

.. py:function:: convert_color(path)

	generate a .h header file from color image at *path*

.. py:function:: convert_bw(path)

	generate a .h header file from B&W image at *path*

.. py:function:: convert_to_raw(path)

	generate a binary .raw file from color image at *path*

.. py:function:: convert_g_to_raw(path)

	generate a binary .raw file from gray image at *path*

.. py:function:: wheel_from_img(path)

	generate a binary .whl file containing wheel position information from gray image at *path*
